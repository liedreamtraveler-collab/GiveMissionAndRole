# 人狼オマージュ型 ロール＆ミッション抽選システム

このシステムは人狼ゲームをオマージュしたリアル／オンライン会話ゲームにおいて、**全員共通のミッション**と**個人ごとのロール**をランダムかつ公平に割り振るWebアプリケーションです。

## 特徴
- リアルタイムでのプレイヤー接続・切断検知 (Socket.io)
- セッション永続化・復帰機能対応 (UUID)
- プリセット登録機能（様々なシチュエーション設定の即時ロード）
- リッチなUI（GSAPによるアニメーション、tsParticlesによる背景エフェクト）

## 動作要件・依存関係
- Node.js (20 LTS 推奨)

## インストール手順

まず初回のみ必要な依存パッケージをインストールします。

```bash
npm install
```

## 起動コマンド

### 開発モード（サーバー/クライアント同時起動）
ホットリロードが有効な状態で起動します。

```bash
npm run dev
```

### 本番起動
サーバーを通して静的ファイルを提供します。
あらかじめフロントエンドのビルドが必要です。

```bash
# クライアントのビルド
npm run build

# サーバー起動 (port 3000)
npm run start
```
起動後、ブラウザから `http://localhost:3000` にアクセスしてください。

## LAN内の他の端末（プレイヤー側）からのアクセス方法

ホストPC（サーバーを起動しているPC）の**ローカルIPアドレス**を確認し、プレイヤー側の端末のブラウザでアクセスしてください。
例: `http://192.168.0.x:3000`

### ローカルIPアドレスの確認方法
- **Windows**: コマンドプロンプトまたはPowerShellで `ipconfig` と入力し、「IPv4 アドレス」の項目を確認。
- **Mac**: ターミナルで `ifconfig` (または `ipconfig getifaddr en0`) と入力して確認。
